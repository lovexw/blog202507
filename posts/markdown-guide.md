# Markdown 写作指南

Markdown 是一种轻量级标记语言，创建于2004年，现在已经成为世界上最流行的标记语言之一。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的HTML文档。

## 为什么使用 Markdown？

- **简单易学** - 语法简单明了，容易掌握
- **纯文本** - 可以用任何文本编辑器编写
- **专注内容** - 让你专注于写作内容，而不是排版
- **兼容性好** - 可以轻松转换为HTML、PDF等格式
- **广泛支持** - 被GitHub、Stack Overflow等众多平台支持
